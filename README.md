# 💳 Profile Card Project

A clean and responsive **Profile Card UI** built using simple web technologies. This project is perfect for beginners learning HTML & CSS, and also useful as a reusable component in larger web apps.

---

## 🚀 Features

* Responsive design (works on mobile & desktop)
* Clean modern UI
* Easy to customize
* Lightweight (no frameworks)
* Uses only HTML + CSS

---

## 📁 Project Structure

```
project-folder/
│── index.html        # Main HTML structure
│── style.css         # Styling and layout
│── code_net.jpg      # Preview image / design reference
```

---

## 🧠 How It Works

The project is divided into two main parts:

### 1. HTML (Structure)

* Creates the card layout
* Includes profile image, name, and description
* Organizes content using semantic tags

### 2. CSS (Styling)

* Adds colors, spacing, and layout
* Uses flexbox for alignment
* Applies shadows and borders for modern look

## 🛠️ How to Use

1. Clone or download the project
2. Open `index.html` in your browser
3. Customize content inside HTML
4. Modify styles in `style.css`
